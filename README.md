1. SSNI / SSIS 系列：S1 No.1 Style

S1 是目前最顶级的AV片商之一，番号从早期的 SSNI 到后来的 SSIS，都代表着顶尖制作水准，画面清晰、女优演出质量高，是很多新手入坑首选。

2. IPX 系列：Idea Pocket
片商：Idea Pocket

风格特点：剧情氛围浓厚，演员颜值与演技兼具，时有制服角色

代表女优：明里紬、桃乃木香奈、相泽南

IPX 系列作品以镜头节奏感强、女主表演真实著称，是老少皆宜的经典片商之一。

3. FSDSS 系列：FALENO
片商：FALENO
